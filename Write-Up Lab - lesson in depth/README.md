# Pembahasan Detail 4 Lab Pentest

Folder ini berisi versi pembelajaran mendalam untuk empat lab:

```text
01_catalina_pembahasan_detail.md
02_portrait_pembahasan_detail.md
03_gazette_pembahasan_detail.md
04_statute_pembahasan_detail.md
```

Setiap file disusun untuk peserta baru dengan struktur:

```text
1. Gambaran umum
2. Data lab
3. Konsep yang dipelajari
4. Fase serangan
5. Command
6. Output yang diharapkan
7. Penjelasan output
8. Troubleshooting
9. Ringkasan hafalan
```

Cara belajar yang disarankan:

```text
Hari 1: baca gambaran umum dan konsep.
Hari 2: jalankan command sambil mencatat output.
Hari 3: ulangi tanpa melihat penjelasan.
Hari 4: coba jelaskan ulang alur ke peserta lain.
```
